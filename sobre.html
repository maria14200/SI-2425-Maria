O que é Multimédia?

Multimédia é a área que engloba o uso de diversas formas de comunicação, como vídeo, áudio, imagem e animação, para criar conteúdos interativos e dinâmicos.

No curso de Multimédia, os alunos aprendem a usar essas diferentes mídias para criar conteúdos inovadores para a web, televisão, cinema e outras plataformas digitais.
